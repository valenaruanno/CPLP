Un error sintactico es aquel que se produce cuando no se cumplen
las reglas de combinacion de componentes basicas y un error 
semantico se produce cuando un programa tiene una estructura
sintacticamente correcta pero su significado o logica es 
incorrecta. 

Ej: var 
        5: Integer;
    
    
    es sintacticamente correcto pero semanticamente incorrecto. 

Ej: int x = 5
    if (x > 3) {
        System.out.println("x es mayor que 3");
    }

    es semanticamente correcto pero no es sintacticamente 
    correcto por que le falta ";" al final de la primer linea.
